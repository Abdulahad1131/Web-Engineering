Portfolio

Project 1: Library Management System

The Library Management System is a software tool designed to help libraries manage their books, members, and transactions efficiently. It allows librarians to handle book cataloging, member registration, and track lending activities. With features like overdue notifications and inventory management, it simplifies daily library operations.

Project 2: Airplane Passenger Satisfaction System

The Airplane Passenger Satisfaction System leverages AI to analyze passenger feedback and enhance the travel experience. By utilizing machine learning algorithms, the system processes survey data and real-time comments to identify trends and sentiment, enabling airlines to make data-driven improvements. This approach helps optimize in-flight services and operational efficiency, ultimately increasing passenger satisfaction and loyalty.

Project 3: Password Manager

The Password Manager is an AI-powered application designed to securely store, generate, and manage user passwords. Utilizing advanced encryption techniques and machine learning algorithms, the system analyzes user behavior to suggest strong, unique passwords and automatically fills in login credentials. By providing real-time security assessments and alerts for potential breaches, the Password Manager enhances user security and simplifies password management.
